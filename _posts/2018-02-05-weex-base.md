## 安装使用

[`weex-toolkit`](https://github.com/weexteam/weex-toolkit) 是官方提供的一个脚手架命令行工具，你可以使用它进行 Weex 项目的创建，调试以及打包等功能。

```bash
npm install weex-toolkit -g --registry=https://registry.npm.taobao.org
```

创建项目

```bash
weex create demo001
```

进入项目

```bash
cd demo001
```

运行项目

```bash
npm run web
```
